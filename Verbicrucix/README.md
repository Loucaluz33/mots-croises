# Verbicrucix

Logiciel de creation de mots croises — application React + TypeScript + Vite.

## Demarrage

```bash
npm install
npm run dev
```

L'application demarre sur `http://localhost:5173` et charge automatiquement `verbicrucix.db`.

## Stack

- React + TypeScript
- Vite (dev server + plugins custom)
- sql.js (SQLite en WASM, cote client)
- Tailwind CSS + shadcn/ui
